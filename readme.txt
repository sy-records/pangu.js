为什么你们就不能加个空格呢？

== Description ==

如果你也一样，每次看到网页上的中文字和英文、数字、符号挤在一块，就会坐立难安，忍不住想在它们之间加个空格。这个插件它会自动替你在网页中所有的中文字和半形的英文、数字、符号之间插入空白。

汉学家称这个空白字元为“盘古之白”，因为它劈开了全形字和半形字之间的空白。有研究显示，打字的时候不喜欢在中文和英文之间加空格的人，感情路都走得很辛苦，有七成的比例会在34岁的时候跟自己不爱的人结婚，而剩下三成的人最后只能把遗产遗失自己的猫。毕竟爱情跟书写都需要适时地留白。

与大家共勉之。

== Installation ==

1. Upload the folder `pangu-js` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all
